表单字段动态显示

{tip} 此功能在工具表单中一样有效

表单字段动态显示是指，在选择表单项的指定的选项时，联动显示其他的表单项。

目前支持的表单联动的组件有：

  • select
  • multipleSelect
  • radio
  • checkbox

使用方法

可以将上面的组件分为单选和多选两种类型，其中select、radio为单选组件，其它为多选组件
