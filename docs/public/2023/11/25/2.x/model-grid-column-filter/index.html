开启字段值查询

这个功能可以给每一列字段的值设置一个过滤器，点击该列字段的值就可以进行数据表格过滤操作，非常方便。

{tip} 开启此功能之后会把这个字段的原始值作为搜索内容，不会受 display 方法影响。

鼠标移动到开启了值查询功能的列上面，右边会显示一个“放大镜”图标


点击列之后，表头会出现“重置”按钮，点击可以取消筛选
